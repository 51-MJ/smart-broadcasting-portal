# 智慧广电后台管理系统

基于 Vue 3 + Vite + Element Plus 的现代化后台管理系统。

## 项目特性

- 🎨 现代化UI设计，使用Element Plus组件库
- 📱 完全响应式设计，支持各种设备
- 🔐 权限管理系统，支持多级权限控制
- 📊 数据仪表板，展示系统数据统计
- 📝 内容管理系统，支持多分类内容发布
- 👥 用户管理系统，支持用户增删改查
- ⚙️ 系统设置，支持邮件、备份、安全配置
- 📋 完整的日志管理功能

## 快速开始

### 前置要求

- Node.js >= 14.0
- npm >= 6.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 项目结构

```
src/
├── api/          # API 服务模块
├── components/   # 可复用组件
├── pages/        # 页面组件
├── router/       # 路由配置
├── store/        # Pinia 状态管理
├── styles/       # 全局样式
├── App.vue       # 根组件
└── main.js       # 入口文件
```

## 主要功能

### 1. 仪表板 (Dashboard)
- 系统数据统计卡片
- 用户增长趋势图
- 内容分类分布
- 最近活动记录

### 2. 内容管理 (Content)
- 内容列表展示
- 搜索和筛选
- 发布新内容
- 编辑和删除

### 3. 用户管理 (Users)
- 用户列表管理
- 添加/编辑用户
- 重置密码
- 启用/禁用用户

### 4. 权限管理 (Permissions)
- 角色权限配置
- 地区权限映射
- 权限编辑

### 5. 系统设置 (Settings)
- 基本设置
- 邮件配置
- 数据备份
- 安全设置
- 日志管理

## 开发建议

1. 使用组件化开发，保持代码组织清晰
2. 使用 Pinia 管理全局状态
3. 使用 Vue Router 进行页面导航
4. 遵循 Vue 3 Composition API 最佳实践
5. 使用 Element Plus 提供的组件和图标
